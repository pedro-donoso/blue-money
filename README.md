# blue-money

*ejecutar desde terminal*

1. Instalar axios:
Se requiere en el proyecto y se generará la carpeta node_modules

```
npm install axios
```

2. Correr archivo
Permitirá crear el archivo de salida con la conversión de divisas

```
node run.js
```
